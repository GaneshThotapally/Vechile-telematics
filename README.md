---

## 📌 Overview

This project is an intelligent Carbon Emission Tracker that predicts vehicle CO₂ emissions using:

- 🚗 Vehicle Telematics Data
- 🧠 LSTM Deep Learning Model
- 📡 Bluetooth ELSM Device
- 📊 Real-Time Charts & Analytics

The system analyzes vehicle parameters like RPM, Speed, Throttle, and Engine Load to predict CO₂ emissions in real time.

## ⚙️ Working Principle

- Vehicle telematics data is collected
- Data includes:
  - RPM
  - Speed
  - Throttle Position
  - Engine Load
- CSV data is normalized and processed
- LSTM model is trained using historical data
- Bluetooth ELSM device connects for live monitoring
- CO₂ emission predictions are generated
- Graphs update dynamically in real time
